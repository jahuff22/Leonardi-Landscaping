# Leonardi Landscaping website concept

A framework-free static site that can be opened with VS Code Live Server and deployed directly to GitHub Pages or Vercel.

## Local preview

Open this folder in VS Code, right-click `index.html`, and choose **Open with Live Server**.

No package installation or build command is required.

## Files

- `index.html` — page content and SEO metadata
- `styles.css` — complete responsive design
- `script.js` — demo quote-form interaction
- `assets/` — hero image, social card, and favicon

## Vercel

Import the GitHub repository in Vercel. Choose **Other** as the framework preset and leave the build command blank. The output directory is the repository root (`.`).

Before a client launch, replace the clearly marked concept content and connect the quote form to an email/form service.
